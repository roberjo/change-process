# Process Diagrams

This directory contains visual representations of our processes and workflows.

## Diagram Standards

### File Formats
- Use `.png` for general diagrams
- Use `.svg` for diagrams that need to be scalable
- Use `.drawio` for editable diagrams (using draw.io)

### Naming Convention
- Use lowercase with hyphens
- Include process name and type
- Example: `change-process-flow.png`

### Diagram Types
1. Process Flows
   - Show step-by-step processes
   - Include decision points
   - Show parallel processes

2. System Architecture
   - Show system components
   - Include data flow
   - Show integration points

3. Decision Trees
   - Show decision points
   - Include outcomes
   - Show escalation paths

4. Timeline Diagrams
   - Show process duration
   - Include dependencies
   - Show parallel activities

## Diagram Structure

### Required Elements
1. Title
2. Legend (if needed)
3. Clear labels
4. Consistent styling
5. Alt text for accessibility

### Style Guide
- Use consistent colors
- Use standard shapes
- Include clear labels
- Maintain proper spacing
- Use readable fonts
